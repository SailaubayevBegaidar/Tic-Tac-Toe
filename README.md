# 🎮 Tic Tac Toe (C# Console Game)

A simple two-player Tic Tac Toe game built using C# in a console application.

---

## 📌 Features

- Two-player mode (Player X vs Player O)
- 3x3 game board
- Input validation (no invalid moves allowed)
- Prevention of overwriting existing moves
- Win detection (rows, columns, diagonals)
- Clean console board rendering

---

## 🧠 Game Rules

- Player 1 uses **X**
- Player 2 uses **O**
- Players take turns entering coordinates:

row col

Example:

1 2


- Rows and columns are indexed from **0 to 2**
- 0 0 → top-left
- 2 2 → bottom-right

---

## 🏆 Win Condition

A player wins if they place three of their symbols in:
- Any row
- Any column
- Any diagonal
